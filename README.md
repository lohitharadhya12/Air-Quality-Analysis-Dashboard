**Air Quality Analysis Dashboard**

**Overview**

This project is an interactive Air Quality Analysis Dashboard designed to analyze AQI levels across major Indian cities. It provides insights into pollution intensity, city-wise air quality comparison, and historical AQI trends to support data-driven environmental analysis.

**Key Features**

Displays average levels of major pollutants: CO, NO, SO2, and PM2.5

Identifies the top 3 most polluted cities and the least polluted cities

Area-wise AQI visualization using geographic maps

Year-wise AQI trend analysis

City-level filtering for focused insights

**Tools and Technologies**

Power BI

Data Visualization

Environmental and AQI Data Analysis


**Project Objective**

To analyze air quality patterns across cities, identify pollution hotspots, and track changes in AQI over time for better understanding and decision-making.
